
# 피그마 Make 기능으로 React 웹사이트 만들기

## 개요
피그마의 Make 기능을 사용하여 프롬프트 입력만으로 React 기반 웹사이트를 생성할 수 있습니다.

## 사용 방법

### 1. Make 기능 시작하기
- 피그마에서 새 파일 생성
- 상단 메뉴에서 **Make** 버튼 클릭
- 또는 `Cmd/Ctrl + /` 단축키 사용

### 2. 프롬프트 입력
```
예시: "React로 랜딩 페이지 만들어줘. 헤더, 히어로 섹션, 기능 소개, 푸터 포함"
```

**효과적인 프롬프트 작성 팁:**
- 구체적인 컴포넌트 명시
- 원하는 스타일/테마 설명
- 색상, 레이아웃 등 세부사항 포함

### 3. 디자인 생성 및 수정
- Make가 자동으로 디자인 생성
- 생성된 디자인 검토
- 추가 프롬프트로 수정 요청

### 4. React 코드 추출
- 생성된 디자인 선택
- **Code** 패널 열기
- React 코드 복사 및 활용

## 주의사항
- 생성된 코드는 기본 구조이므로 커스터마이징 필요
- 반응형 디자인은 별도 프롬프트로 요청
- 복잡한 기능은 단계별로 생성 권장

# Figma Make 에서 Edit 로 디자인 가져오기
1. 우측 상단 디자인 복사 버튼 클릭
2. Edit 탭으로 이동
3. 디자인 붙여넣기 (Ctrl + V)
